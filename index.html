Information Visualization

MBA 677 / BUS 377: Summer, 2014



Welcome to the course

You will learn why visualization matters and how to represent data visually and truthfully.
You might get pretty good at Tableau along the way.

Things we use regularly, starting from day one. See the resources section for more details.

A book

Show Me the Numbers: Designing Tables and Graphs to Enlighten (2nd edition), by Stephen Few - ISBN: 0970601972, Analytics Press 2012. This book is available in the bookstore.

Piazza

Piazza is used for all class discussions and announcements. You will need to create an account (free) if you don't already have one.

Tableau

Tableau Desktop is the commercial visualization software we use throught the semester. It is normally $1999 per user it is free to you for the duration of this class through the Tableau for teaching program.

Blackboard

As much as I hate Blackboard, it is a good vehicle for storing grades and giving quizzes. That is all we will be using it for.

Frequently Asked Questions


I'm not good with technology, is this the class for me?

No it isn't. We will be using different tools that require installation and are not available in the lab. I can't come to your house to help you but will be happy to respond, as best I can, to your email queries.

Why don't you put this entire course in Blackboard?

I could write a lengthy post on this, but I won't. In my opinion, Blackboard is not a good tool for online discussions and makes it difficult to create an intuitive flow for a course.

Is there a mac version of Tableau?

The mac version of Tableau was released with the 8.2 release on June 19th, 2014. Your student key will work for either the mac or windows version.

Can I complete assignments late?

No, not ever. It kills the discussion format of this class.

Why are we using Tableau instead of Excel?

In my opinion, Tableau makes better visualizations than Excel by default. Especially interactive visualizations that you can share on the web.

How did you design this course website?

I modified a Twitter Bootstrap theme called Progressus. It is licensed by Sergey Pozhilov under a creative commons attribution license. This is the attribution.
